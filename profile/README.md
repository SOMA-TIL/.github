### ✨ Project Main Feature

#### 1️⃣ 기술 문제 풀이 기능

> 기술 면접에서 자주 등장하는 문제들을 체계적으로 학습하여 연습할 수 있도록 합니다.

#### 2️⃣ 모의 기술 면접 기능

> 사용자 포트폴리오 분석 및 기업 맞춤형 모의 면접 & 카테고리 기반 모의 면접을 통해  
> 효과적으로 기술 면접에 대비할 수 있도록 합니다.
